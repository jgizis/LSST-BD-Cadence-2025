# LSST-BD-Cadence-2025
revisiting the LSST brown dwarf parallax performance with 2025 sims.

remember to install rubin-sim (which includes rubin-scheduler)
See https://rubin-sim.lsst.io

See my repo LSST-BD-Cadence for the original calculations which 
were published in Gizis et al., 2022, ApJS, 263, 23
